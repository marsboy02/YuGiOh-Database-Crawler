# YuGiOh-Database-Crawler

유희왕 카드 DB를 크롤링하여 카드의 정보를 알려주는 API입니다.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# using docker
$ docker-compose up
$ docker-compose down
```
